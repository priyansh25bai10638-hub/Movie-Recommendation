# 🎬 Movie Recommender System

An interactive Streamlit web app that helps you discover movies by filtering through genre, IMDb rating, studio, and director. Say goodbye to decision fatigue!

## ✨ Features
- Filter by multiple genres, minimum rating (0-10), studio, and director
- Randomly picks from all matching movies to eliminate bias
- Beautiful UI with instant recommendations and balloon animations 🎈
- Curated database of 100+ critically acclaimed films across all tiers

## 🚀 Quick Start
```bash
pip install streamlit
streamlit run app.py
